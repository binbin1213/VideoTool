   - ✅ 整体视觉协调

#### 🌍 测试语言切换

1. **在"偏好设置"中找到"界面语言"**
   
2. **切换语言**

3. **观察哪些内容已翻译**
   - ✅ 侧边栏菜单（全部）
   - ✅ FFmpeg 提示框（如果显示）

1. **字体变化**

## 🐛 可能遇到的问题

### 问题1：字体看起来变大了
**正常现象！** 这是设计规范要求的。
- 旧版本：11px
- 新版本：14px（+27%）
- **目的**：提升可读性，符合桌面应用标准

**如果觉得太大**，可以临时调整：
```scss
// src/renderer/styles/tokens.scss
$font-size-base: 13px; // 或 12px
```

### 问题2：深色主题看不清楚
**可能原因**：
1. 某些组件还未适配深色主题
2. 对比度不够

**解决方案**：
- 这是正常的，我们只更新了核心样式
- 后续会逐步优化所有组件
- 暂时可以使用浅色主题

### 问题3：部分文案还是中文/英文
**正常现象！** 我们只翻译了核心组件：
- ✅ 已翻译：侧边栏、FFmpeg提示、日志
- ⏳ 未翻译：各功能 Tab 内容

**下一步**：会继续翻译其他组件

## 🎓 开发者提示

### 查看主题状态
打开浏览器控制台（F12）：
```javascript
// 查看当前主题
localStorage.getItem('vt-theme-preference')

// 查看当前语言
localStorage.getItem('vt-language')

// 强制设置主题
localStorage.setItem('vt-theme-preference', 'dark')
location.reload()
```

### 清除所有设置（重置）
```javascript
localStorage.clear()
location.reload()
```
